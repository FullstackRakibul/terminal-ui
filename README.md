# terminal-ui

Open Git Bash in your terminal.

Type ```nano ~/.bashrc ``` and press Enter.

Add this line at the very bottom (replace the path with where you saved your file):

Bash
```
eval "$(oh-my-posh init bash --config 'C:/Users/YourName/Documents/mytheme.omp.json')"

```
Press Ctrl+O, then Enter to save, and Ctrl+X to exit.

Restart your Terminal.
